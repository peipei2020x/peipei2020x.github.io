# PERING - Official Portal

**PERING** は、peipeiが創造するプロジェクト群への入り口となるポータルサイトです。
単なるリンク集にとどまらず、ユーザー自身がカスタマイズ可能なダッシュボード機能を備えています。

## 🚀 特徴

このポータルは **単一のHTMLファイル (`index.html`)** だけで動作し、以下の機能を内包しています。

### デザイン & 基本機能
- **レスポンシブデザイン**: PC、タブレット、スマートフォンに対応。
- **テーマ切り替え**: ライトモード / ダークモード（設定は自動保存）。
- **ビジュアルエフェクト**: グラスモーフィズム（すりガラス風）デザインと、カーソル追従オーラエフェクト。
- **ユーティリティ**: リアルタイム時計、Google検索バー。

### 🛠 隠し機能 (Advanced Features)

このサイトには、特定の操作を行わないと表示されない「設定パネル」が存在します。

#### 🔓 設定パネルの開き方
画面左上のロゴ **「PERING」を 7回連続でクリック** してください。
設定パネルがオーバーレイ表示され、以下の機能が利用可能になります。

1. **表示設定**:
   - 時計や検索バーの表示/非表示切り替え。
2. **リラックスビュー (Relax View)**:
   - お気に入りの画像や動画(MP4等)を背景としてオーバーレイ表示する機能。
   - **IndexedDB** を使用し、ブラウザ内にメディアファイルを保存するため、次回アクセス時も設定が保持されます。
   - 起動トリガー（キー長押し or トグル）や、特定のキー割り当てが可能。
3. **カスタムショートカット**:
   - よく使うサイトへのリンクを自由に追加・削除できます（LocalStorageに保存）。
4. **メモ帳**:
   - ブラウザに保存されるシンプルなメモ帳機能。

## 💻 技術スタック

外部ライブラリやフレームワークを一切使用しない、ピュアな構成です。

- **HTML5**
- **CSS3** (Variables, Grid, Flexbox, Animations, Backdrop-filter)
- **JavaScript (Vanilla)**
  - `LocalStorage`: テーマ、ショートカット、メモの保存
  - `IndexedDB`: リラックスビュー用のメディアファイル（Blob）保存

## ⚠️ データ管理について

- **プライバシー**: 入力されたメモや設定、アップロードした画像・動画は、すべて**閲覧者のブラウザ内（ローカル環境）にのみ保存**されます。サーバーへ送信されることはありません。
- **データリセット**: ユーティリティセクションにある「全設定をリセット」ボタン、またはブラウザのキャッシュクリアで初期化されます。
